# M9A Update Assistant

M9A CLI 更新小助手

## 如何使用

1. 从 [Release](https://github.com/NEANC/M9A-Update-Assistant/releases/latest) 下载
2. 首次运行会生成 `config.ini` 配置文件
3. 编辑 `config.ini` 文件，配置 M9A 文件夹路径、临时文件夹路径和代理服务器信息
4. 运行程序，开始更新
